excerpt: "
We are working on a model that can generate commonsense inferences on provided image + textual cues. 
For this, we are extending VisualCOMET to reason about entities  (e.g.: Oven is used for baking) in addition to events (e.g.: Before shooting, the person loaded a gun).
This will be developed by using existing Vision language sources rather than collecting expensive human annotations for each image."
